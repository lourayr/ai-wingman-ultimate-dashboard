AII-GPT-Question-Summaries
Looking at each GPT's actual job and the sample questions provided, here are the distilled intake questions, 5-7 each, in compressed prompt-ready form:

---

## 🎯 Biz Strategy Agent

1. Business name, what you do, who you serve
2. Current revenue stage + team size
3. Core offers (2-3), what's working/stalling
4. Single most important goal right now
5. Real constraints: budget, hours, capacity
6. Win definition: 30/60/90 day metrics
7. 12-24 month vision + desired lifestyle/role

---

## 🤝 Private Strategic Partner GPT

1. Mode: Diagnose / Campaign / Automate / Implement / Review
2. Session outcome: single most important deliverable
3. Ideal client: daily reality, what's failed them
4. Your one promise + unique mechanism/process name
5. Offer: format, delivery, price tiers
6. Top 3 audience objections/beliefs to address
7. Channels + smallest executable next step this week

---

## 🛠️ ArchitectGPT

1. GPT name + role + who it serves
2. Problem it solves + public/private/internal use
3. Operating modes (e.g., Diagnose, Build, Review)
4. Output per mode + 3-5 non-negotiable rules
5. Tone: emotional register, phrases to avoid
6. Core domain doctrine/frameworks to hardcode
7. Deployment format: JSON / text blocks / system prompt

---

## ⚡ Offer Stack Agent

1. Single transformation delivered + niche
2. Ideal buyer: age, situation, gap (specific, not categorical)
3. Core problem + urgency + cost of inaction
4. 3-5 fears + 3-5 limiting beliefs buyer holds
5. Dream outcome + why current solutions fail
6. Unique method/mechanism + existing results
7. Price target + launch timeline + top objection heard

---

## 🤖 Workflow Engineering Copilot

1. Workflow name + measurable success outcome
2. Trigger event + every step start to finish
3. Tools/apps used at each step
4. Where AI could help + edge cases/exceptions
5. Preferred platform: Make / Zapier / n8n
6. Compliance/sensitivity constraints
7. Timeline + implementation skill level

---

## ⚙️ AutoFlow Architect

1. Process to automate (one sentence)
2. Trigger: what starts it
3. Steps from trigger to done
4. Tools currently involved
5. Preferred platform: Make / Zapier / n8n
6. Biggest manual pain point right now
7. Go-live timeline

---

## 🧠 AI Agent Selection Agent

1. Top 1-3 job categories for the agent
2. Where it lives: Slack / email / CRM / dashboard
3. Autonomy level: suggest / draft / execute / autonomous
4. Top 5 workflows it must own (trigger to done)
5. Data sensitivity + tools it must connect to
6. Dashboard needs: top actions, risks, opportunities daily
7. Budget ceiling + timeline + who owns maintenance

---

## ✍️ AI-Optimized Blog Creation GPT

1. Primary goal: SEO / leads / authority / sales
2. Ideal reader: role, industry, awareness level (cold/warm/hot)
3. Topic + your unique angle / proprietary material
4. Format + length + tone preference
5. Primary CTA + secondary CTA
6. Keywords, internal links, offers to support
7. Single post or content system + sustainable cadence

---

## 🤝 JV Partner Kit Agent

1. Business name + what you do + who you serve
2. Your methodology/framework name (if exists)
3. What you're launching + full offer stack + price tiers
4. Revenue goal: next 90 days
5. Ideal JV partner: audience type + size
6. Commission structure + campaign dates
7. Brand voice + one core story + client results/proof

---

## 🗺️ Customer Journey Architect

1. Problem you solve + who specifically
2. Your core transformation (one sentence)
3. Current offers + price points
4. Biggest gap in your current offer ladder
5. Ideal client's next logical step after working with you

---

## 📊 Data Helper Planning Agent

1. Business type + primary revenue model
2. Current tools collecting data (CRM, forms, email, etc.)
3. Data goal: analytics / AI training / personalization / reporting
4. Biggest decision you wish data could answer
5. Current data gaps you already know about
6. Team capacity for data work: none / some / dedicated
7. Timeline + data maturity self-assessment (1-10)

---

## 🧹 Data Collection & Cleaning Agent

1. AI use case: chatbot / fine-tuning / analytics / automation
2. Data you already have + format (CSV, PDF, CRM export, etc.)
3. Data you need but don't have yet
4. Quality issues: duplicates, inconsistencies, missing fields
5. Labeling needs: categories, sentiment, intent, other
6. Output destination: which AI tool or platform
7. Timeline + volume of data involved

---

[GPT Name] Intake:
1. [compressed question]
2. [compressed question]
